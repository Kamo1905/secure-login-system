# Secure Login System 
A Python-GUI authentication system that securely register and log in users using password hashing. 

## Features 
- User registeration and login
- Password hashing using SHA-256
- Password strength validation
- Show/Hide password functionality
- Simple user dashboard

## Technologies Used 
- Python
- Tkinter
- SqLite
- hashlib (security)
- Regualar Expression

## How to Run
python main.py

## Purpose 
This project demostrates basic cybersecurity principles such as secure password storage and authentication systems.
